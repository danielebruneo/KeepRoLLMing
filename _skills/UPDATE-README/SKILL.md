# UPDATE-README Skill

## Goal
Update [README.md](CATALYST.md) without disturbing stable sections unnecessarily.

## Procedure
1. Read the current [README.md](CATALYST.md) fully.
2. Identify what actually changed in the project.
3. Update only the sections impacted by the change.
4. Keep the README human-oriented.
5. Avoid injecting task-local implementation notes.
