# ADD-FEATURE Skill

## Goal
Implement a requested feature in a controlled, test-aware way.

## Procedure
1. Create or refresh [_agent/ACTIVE_TASK.md](../../_agent/ACTIVE_TASK.md).
2. Identify the minimal set of modules involved.
3. Implement the feature with minimal collateral change.
4. Add or update tests if appropriate.
5. Record the outcome in [_agent/HANDOFF.md](../../_agent/HANDOFF.md).
