# Done Criteria

A task is considered done when:
- the requested outcome is implemented or the requested analysis is completed
- relevant checks or tests were run when applicable
- scope stayed aligned with the task
- [_agent/HANDOFF.md](HANDOFF.md) is updated
- any non-obvious lesson is captured in [_agent/MEMORY.md](MEMORY.md)

## Template Compliance
- All documentation files created or modified follow established template formats.
- Cross-referencing between related documentation elements and skills is maintained.
