# IMPROVE-DOC Skill Directory

This directory contains the IMPROVE-DOC skill, which helps extend and improve documentation in the ReBadge.API project where it makes sense.

## Overview
The IMPROVE-DOC skill provides guidance for:
1. Identifying opportunities to enhance existing documentation
2. Creating new documentation files when needed
3. Improving cross-referencing between documents
4. Maintaining consistency with CATALYST framework conventions

## Files in This Directory
- `SKILL.md` - The main skill definition and usage guide

## Usage
When you need to extend or improve project documentation, use the IMPROVE-DOC skill following the guidelines outlined in SKILL.md.
