# PROPOSE-CATALYST-PULL-REQUEST

## Goal
Prepare a structured PR-style proposal for a change that should be reviewed for inclusion in the upstream CATALYST core.

## Expected sections
- Title
- Problem
- Local evidence
- Current local solution
- Why this belongs in core
- Proposed core changes
- Compatibility and migration notes
- Recommendation
