# CONSOLIDATE-DOC Skill Directory

This directory contains the CONSOLIDATE-DOC skill, which helps consolidate documentation in the ReBadge.API project by identifying and removing duplicates, organizing content logically, and avoiding unnecessary fragmentation.

## Overview
The CONSOLIDATE-DOC skill provides guidance for:
1. Identifying duplicate or redundant information across files
2. Merging overlapping content into single authoritative sources
3. Consolidating related topics that should be grouped together
4. Removing obsolete or outdated documentation sections
5. Creating new organized structures when needed

## Files in This Directory
- `SKILL.md` - The main skill definition and usage guide

## Usage
When you need to consolidate documentation, use the CONSOLIDATE-DOC skill following the guidelines outlined in SKILL.md.
